Introduction

Greedy algorithm method is one of the strategy like Divide and Conquer which is used to solve the problems. It is used for solving Optimization problem.
An optimization problem is a problem is a problem that demands either maximum or minimum choice.
It is an straight forward approach.

Function

The main function of this approach is that decision is taken on the basis of current available information. Whatever the current information is present , the desicion is made without worrying about the effect of current decision in future.

Components of Greedy Algorithm

  1. Candidate Set : A solution that is created from set is known as a Candidate set.
  2. Selection function : This function is used to choose the candidate or subset which can be added in solution
  3. Feasibility function : A function that is use to determine whether candidate or subset can be used to contribute to solution or not.
  4. Objective function : A function is used to assign the value to the solution or partial solution.
  5. Solution function : It is used to initiate whether the complete function has been reached or not.
